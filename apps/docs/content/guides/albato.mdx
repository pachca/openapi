# Albato

[Albato](https://albato.ru/) — коммерческая платформа-посредник для интеграции **250+ сервисов** с Пачкой без написания кода.

## Принцип работы

Триггер в стороннем сервисе (новый заказ в CRM, коммит в репозитории, оплата) → действие в Пачке (отправка сообщения в чат).

<Info>Пачка обрабатывает только входящие события — получает данные, но не отправляет обратно через Albato.</Info>

## Возможности

- **Текстовые уведомления** с гибким форматированием
- **Условная логика** — фильтры для отправки только нужных событий
- **Неограниченное количество** интеграций в одном чате
- Создание доступно только **администраторам**

## Поддерживаемые сервисы

Битрикс24, AmoCRM, СБИС, Jira, Trello, МойСклад, CloudPayments, YClients, Telegram и [другие](https://albato.ru/integrations).

## Стоимость

14 дней бесплатно, далее платная подписка на сайте Albato.
