# Markdown и документы

Агент может отдавать результат не «простынёй» текста, а оформленным документом. Пачка понимает Markdown и в сообщениях, и в прикреплённых `.md`-файлах. Отдельная интеграция для этого не нужна.

## Markdown в сообщениях

Текст сообщений — Markdown (CommonMark с расширениями GFM: автоссылки, зачёркивание, упоминания). Жирный, курсив, списки, цитаты, инлайн-код и блоки кода, ссылки и упоминания работают сразу — агенту достаточно прислать обычный Markdown в тексте сообщения.

Разметка передаётся в поле `content` метода [Новое сообщение](POST /messages) как есть — отдельный флаг включать не нужно, текст всегда интерпретируется как Markdown. Те же правила действуют при [редактировании сообщения](PUT /messages/{id}).

## Файл md рендерится карточкой

Если бот прикрепляет к сообщению `.md`-файл, Пачка показывает его не ссылкой на скачивание, а оформленной карточкой прямо в чате с разворотом в полноэкранный просмотр. Поддерживается:

- заголовки, списки, цитаты, ссылки, изображения
- таблицы и чек-листы задач
- блоки кода с подсветкой синтаксиса
- блоки `diff` с подсветкой добавленных и удалённых строк
- диаграммы `mermaid`

Всё, что присылает бот, рендерится так же, как контент от людей. Специального режима для агентов не требуется.

Загрузите `.md` по схеме [Загрузка файлов](/api/file-uploads) (три шага через presigned S3), затем передайте полученный ключ файла в поле `files` при вызове [Новое сообщение](POST /messages). Расширение `.md` в имени файла включает рендер карточкой автоматически.

## Возможности и кейсы

- **Отчёты и сводки** — агент собирает данные и присылает структурированный отчёт с таблицами вместо стены текста
- **Код-ревью и диффы** — агент прикладывает `.md` с блоками `diff`, изменения видны зелёным и красным
- **Спецификации и заметки встреч** — оформленный документ в треде, обсуждение рядом
- **Диаграммы** — последовательности и схемы через `mermaid` прямо в карточке

## Ограничения

- Текст сообщения — до 40 КБ Markdown
- Предпросмотр `.md`-файла рассчитан на типовой размер документа. Очень крупные файлы открываются как обычное вложение для скачивания
