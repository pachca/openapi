
# Интеграция с Albato

Albato — это платформа, которая помогает интегрировать между собой разные программы (CRM, системы аналитики, мессенджеры и т.д.). Через **Albato к Пачке** можно подключить более 250 сервисов и получать [уведомления](https://pachca.com/help-center/features/kakie-est-uvedomleniya-v-pachke) прямо в чаты и [каналы](https://pachca.com/help-center/features/types-of-chats) Пачки.

Уведомления будут приходить в виде [сообщения](https://pachca.com/help-center/features/actions-with-messages) от того, кто подключил интеграцию. Количество интеграций в одном чате не ограничено.

> Пачка пока способна обрабатывать только входящие события. Поэтому у вас не получится прислать информацию из Пачки другим сервисам.


> **Внимание:** Albato работает на коммерческой основе. [Тарифы](https://pachca.com/help-center/prices/how-to-choose-a-plan) можно найти на [сайте](https://albato.ru/pricing) сервиса, первые 14 дней — бесплатно.


## Какие сервисы можно подключить через Albato

Через Albato к Пачке можно подключить более 250 сервисов.

Так, например, можно подключить к Пачке вашу CRM (Битрикс24, AmoCRM, СБИС CRM) и получать уведомления о продажах/подписках. Или настроить получение уведомлений из вашего сервиса для управления проектами (Jira, Trello), чтобы понимать, какие изменения там происходят.

Еще среди сервисов, доступных для подключения, можно найти МойСклад, CloudPayments, YClients и др. Полный список можно посмотреть при создании связки в личном кабинете Альбато (об этом ниже).

## Какие уведомления будут приходить в Пачку

Через Albato можно настроить текстовые уведомления о разных событиях. Какие это события — зависит от конкретного сервиса. Например, при интеграции с Битрикс24 в Пачку могут приходить уведомления об изменении в сделке, создании счета, новом лиде и тд. А при интеграции с Telegram — о входящем сообщении или новом участнике группы.

Причем текст сообщения, которое будет отправляться в чат, можно настроить вручную. Например, если вам пришло сообщение в Telegram, в Пачку можно отправить просто текст этого письма, а можно добавить пояснение (например, «в Telegram пришло сообщение») и указать ник пользователя.

## Кто может создавать интеграции через Albato

Интеграцию через Albato может создавать только [администратор](https://pachca.com/help-center/features/system-roles).

## Как настроить интеграцию через Albato


  ### Шаг 1. Зарегистрируйтесь в Albato

![Форма регистрации на сайте Albato](/images/albato/registration.png)

*Форма регистрации на сайте Albato*


    1. Откройте [сайт](https://albato.ru/) Albato и нажмите на кнопку «[Регистрация](https://pachca.com/help-center/start/signup-signin)» в верхнем правом углу
    2. Введите свои данные и нажмите «Зарегистрироваться»
    3. Проверьте почту, вам должен прийти код подтверждения
    4. Введите код


  ### Шаг 2. Подключите Пачку к Albato

![Раздел подключений в личном кабинете Albato](/images/albato/connections.png)

*Раздел подключений в личном кабинете Albato*


    ![Выбор Пачки в списке сервисов Albato](/images/albato/select-pachca.png)

*Выбор Пачки в списке сервисов Albato*


    1. Зайдите в свой [аккаунт](https://pachca.com/help-center/administration/about-accounts) в Пачке в том браузере, в котором вы собираетесь настраивать интеграцию
    2. Вернитесь на сайт Albato и в личном кабинете откройте раздел «Подключения»
    3. Найдите кнопку «Добавить подключение» в верхнем левом углу и нажмите на нее
    4. Выберите в списке сервисов Пачку
    5. Добавьте название подключения либо оставьте базовое и нажмите «Далее»
    6. Нажмите «Предоставить доступ Albato»
    7. В открывшемся окне нажмите «Продолжить»


  ### Шаг 3. Подключите сервис, из которого хотите передавать данные

![Добавление нового подключения к сервису](/images/albato/add-connection.png)

*Добавление нового подключения к сервису*


    ![Настройка подключения внешнего сервиса](/images/albato/setup-connection.png)

*Настройка подключения внешнего сервиса*


    1. В личном кабинете откройте раздел «Подключения» и нажмите «Добавить подключение»
    2. Выберите в списке сервис, который хотите интегрировать с Пачкой, и нажмите «Предоставить доступ»
    3. Добавьте название подключения либо оставьте базовое, введите необходимые данные и нажмите «Далее»

    Для подключения разных сервисов нужны разные данные, в базе знаний Albato есть [инструкции](https://blog.albato.ru/category/instrukczii/), как подключить основные сервисы.


  ### Шаг 4. Добавьте триггер для создания связки

![Создание триггера для связки в Albato](/images/albato/create-trigger.png)

*Создание триггера для связки в Albato*


    1. В личном кабинете перейдите в раздел «Мои связки» и нажмите «Создать новую связку» в левом верхнем углу
    2. Нажмите на кнопку «Добавить триггер, который будет запускать связку»
    3. Выберите сервис, из которого будут передаваться данные в Пачку
    4. Выберите событие, которое будет запускать передачу данных
    5. Выберите подключенный аккаунт
    6. Нажмите «Добавить триггер»
    7. При необходимости укажите дополнительные настройки (для каждого сервиса они будут свои)

    > В Albato можно добавлять условия для прерывания или продолжения работы связки (то, в каких случаях данные будут передаваться, а в каких — нет). Это можно сделать через фильтр входящих данных, который находится внутри кнопки с триггером. Подробнее о нем — в [статье](https://blog.albato.ru/instrument-filtr-vhodyashhih-dannyh/) Albato.


  ### Шаг 5. Добавьте действие, которое будет происходить после старта связки

![Добавление действия после триггера](/images/albato/add-action.png)

*Добавление действия после триггера*


    ![Настройка отправки сообщения в Пачку](/images/albato/send-message.png)

*Настройка отправки сообщения в Пачку*


    1. Нажмите на кнопку «Добавьте действие, которое будет происходить после старта связки»
    2. Выберите Пачку в разделе «Сервис»
    3. Выберите «Отправить сообщение» в разделе «Действие»
    4. Выберите подключенный аккаунт
    5. В открывшемся окне введите ID чата/канала, в который вы хотите отправлять уведомления. Найти ID чата/канала проще всего в адресной строке браузерной версии Пачки, он состоит из семи цифр и располагается после «chats/»
    6. В разделе «Тип чата» выберите «Беседа»
    7. Создайте текст сообщения из данных, которые вы хотели бы передать в Пачку, и, возможно, дополнительного текста
    8. Нажмите сохранить


  ### Шаг 6. Запустите связку

Осталось только запустить связку, нажав на кнопку в нижнем правом углу. Как работают запущенные связки, можно посмотреть в личном кабинете в журнале связок.
