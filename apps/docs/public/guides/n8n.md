
# n8n

[n8n](https://n8n.io/) — визуальный конструктор автоматизаций с открытым исходным кодом. С помощью Пачка-нода для n8n вы можете создавать автоматизации без написания кода.

> **Внимание:** Интеграция с n8n находится в статусе бета-версии.


## Принцип работы

Автоматизации в n8n строятся из узлов, соединённых в цепочки:

- **Узлы-триггеры** запускают процесс (событие, расписание, нажатие)
- **Узлы-действия** выполняют операции (отправить сообщение, создать пользователя)
- **Цепочки узлов** формируют полноценные workflow

## Поддерживаемые операции

| Категория | Операции |
|-----------|----------|
| **Сообщения** | Отправка, редактирование, удаление, закрепление |
| **Треды** | Создание, получение |
| **Реакции** | Добавление, удаление |
| **Чаты** | Создание, архивирование, управление участниками |
| **Пользователи** | Создание, редактирование, удаление |
| **Теги** | Управление группами |
| **Статусы** | Установка профиля |
| **Формы** | Открытие и обработка |

## Развёртывание

n8n развёртывается на собственном сервере через Docker или Node.js:

**Docker**

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n
```


## Настройка

Для подключения Пачки в n8n создайте credentials с двумя параметрами:

| Параметр | Значение |
|----------|----------|
| **Access Token** | Токен бота или персональный токен. Получить: [Авторизация](/api/authorization) |
| **Base URL** | `https://api.pachca.com/api/shared/v1` |

> Подробнее о настройке — в статье
  [Автоматизации в Пачке с помощью n8n](https://pachca.com/help-center/integrations/avtomatizacii-v-pachke-s-pomoshchyu-n8n)
