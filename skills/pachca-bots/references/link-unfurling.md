### Разворачивание ссылок (unfurling)

1. Создай специального Unfurl-бота и укажи отслеживаемые домены

2. При появлении ссылки бот получает вебхук `"event": "link_shared"` с массивом `links`. Каждый объект содержит `url`, `domain` и `skip: boolean` — если `skip` равен `true`, автор скрыл превью и бот должен пропустить эту ссылку

3. Извлеки данные из своей системы по URL из `links` (пропуская объекты с `skip: true`)

4. Отправь превью-данные:
   ```bash
   pachca messages unfurl <message_id> --link-previews='{"https://example.com":{"title":"Example","description":"Description"}}'
   ```

> Эндпоинт привязан к конкретному сообщению. Необходим специальный Unfurl-бот с указанными доменами.
